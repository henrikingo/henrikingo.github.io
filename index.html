@henrikingo's github pages

A couple of my github projects I've published as nice demos:

  • My fork of impress.js
    • Impress.js is a framework for creating cool 3D presentations (not slides!) with HTML5 and CSS3.
    • With spiritual guidance from the original creator of this cool software, @bartaz, I've added a plugin api and some new functionality as plugins.
    • I also added another demo presentation. That one actually looks like old fashioned slides.
    • A third demo presentation shows how to use inline Markdown to author a Impress.js presentations. (Another feature I added.)
    • 3D-rotations demo showcases movements truly in a 3D space.
    • The cube is another demo of using Impress.js in 3D space, using the simple form of a 6-sided cube or dice. I intend to add some cool features to it later.
  • impress-templates contains templates you can use to get started with impress.js presentations.
  • Impressionist is a 3D visual editor to create impress.js presentations.
  • A repository of real presentations I've done at conferences.

I have several other projects on github as well. You'll have to check out their README and source code, they don't have nice project pages.
